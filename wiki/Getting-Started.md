# Getting Started

You don't need to install anything to try Covaga ECAD Viewer — it runs in a
regular web browser.

## 1. Open the viewer

Go to **[covaga.dev](https://covaga.dev)** (or run it locally — see
[[Installing Covaga ECAD Viewer]]).

## 2. Open a file

There are three ways to load content:

- **Drag & drop** a `.epdz` project or a `.e3d` part onto the window.
- **Open file** — top-right button, pick a `.epdz` or `.e3d` from disk.
- **Sample files** — open **Settings** (the ⚙ gear at the bottom of the left
  rail) → *Sample files* → **Sample project** or **Sample 3D part**. This is the
  fastest way to see what the viewer does without exporting anything.

> Don't have a `.epdz` yet? See [[Exporting a Project from EPLAN]].

## 3. Find your way around

The left **rail** switches between the main views:

| Icon | View | Shows |
| --- | --- | --- |
| 🏠 | **Home** | Your library and recent files. |
| ⬚ | **3D** | The 3D installation spaces of the project. |
| ▤ | **Schematics** | The 2D schematic pages. |
| ⓘ | **Project** | Project metadata read from `manifest.db`. |

When you open a `.epdz` that contains 3D, the viewer jumps straight into the
**3D** view. Use the model selector (top-left) to switch between installation
spaces, and the **ISO / Front / Side / Top** presets (bottom-left) to reframe
the camera.

See [[Navigating the Viewer]] for the full tour.

## A note on privacy

Everything is processed **on your device**. No file, page or model is ever
uploaded to a server — you can even use the app offline once it's installed as a
[[Installing Covaga ECAD Viewer|PWA]].
